## Ví dụ Hotel Booking Bot

### 1. Conversation Flow
```text
User: "I want to book a hotel"
Bot: "Which city would you like to stay in?"
User: "New York"
Bot: "When would you like to check in?"
User: "Next Friday"
Bot: "How many nights?"
User: "3 nights"
Bot: [Invokes Lambda] "Your booking is confirmed..."
```

### 2. Intent Configuration
```json
{
  "intent": "BookHotel",
  "slots": {
    "city": {
      "type": "AMAZON.City",
      "required": true
    },
    "checkInDate": {
      "type": "AMAZON.DATE",
      "required": true
    },
    "duration": {
      "type": "AMAZON.NUMBER",
      "required": true
    }
  }
}
```
